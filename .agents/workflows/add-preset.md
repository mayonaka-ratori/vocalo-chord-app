# コード進行プリセット追加

## ステップ

1. **プリセット情報の確認**
   ユーザーから以下を確認する:
   - プリセット名（日本語・英語）
   - 度数表記でのコード進行（例: IV V IIIm VIm）
   - 雰囲気タグ
   - 参考楽曲名

2. **データ追加**
   - `data/presets.ts` に新しいプリセットオブジェクトを追加
   - id は kebab-case で一意にする
   - degrees は8小節分（4小節パターンの場合は2回繰り返す）

3. **全キーで検証**
   - 新プリセットが C, G, D, F, Bb, Eb, A, E の全キーで正しく移調されることを確認
   - `lib/music/transpose.ts` のテストを実行

4. **UI検証**
   - ブラウザでプリセットカードが正しく表示されることを確認
   - タグフィルタリングで正しく絞り込まれることを確認
   - 選択時にコード進行が正しく入力されることを確認
   - 再生して音が正しいことを確認
