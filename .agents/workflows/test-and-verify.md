# テスト・検証ワークフロー

## ステップ

1. **ビルド確認**
   ```bash
   npm run build
   ```
   エラーが無いことを確認する

2. **型チェック**
   ```bash
   npx tsc --noEmit
   ```

3. **Lint**
   ```bash
   npm run lint
   ```

4. **ブラウザ検証（PC）**
   - http://localhost:3000 を開く
   - 全プリセットが正しく表示されるか確認
   - コード進行の再生が正しく動作するか確認
   - MIDI書き出しが正しく動作するか確認
   - スクリーンショットを撮影する

5. **ブラウザ検証（モバイル）**
   - Chrome DevTools でモバイルビューに切り替える（iPhone 14 Pro サイズ）
   - タップ操作が正常か確認
   - フローティング再生バーが表示されるか確認
   - スクリーンショットを撮影する

6. **検証結果のウォークスルー作成**
   - 確認した内容と結果をウォークスルーにまとめる
