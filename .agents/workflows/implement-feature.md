# 機能実装ワークフロー

## ステップ

1. **要件整理**
   - 機能の目的と、どのペルソナのどの課題を解決するかを明確にする（@PERSONA.md 参照）
   - 影響範囲（変更するファイル）をリストアップする

2. **型定義**
   - 新しい型が必要な場合、まず `types/` に型定義を追加する

3. **データ層**
   - 新しいデータが必要な場合、`data/` にデータファイルを追加する

4. **ロジック層**
   - `lib/` に必要なロジック関数を実装する
   - 純粋関数として実装し、副作用を分離する

5. **ストア更新**
   - `lib/store.ts` に必要な状態とアクションを追加する

6. **UIコンポーネント**
   - `components/` にコンポーネントを作成する
   - レスポンシブ対応を含めて実装する（mobile-first）

7. **ページ統合**
   - `app/page.tsx` に新コンポーネントを配置する

8. **ブラウザ検証**
   - PC とモバイルの両方で動作確認する
   - Antigravity Browser でスクリーンショットを撮影する
