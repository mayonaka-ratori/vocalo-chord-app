# リズムパターン追加

## ステップ

1. **パターン情報の確認**
   - パターン種類: drum / bass / backing
   - パターン名（日本語）
   - パターン説明
   - ステップシーケンスデータ

2. **データ追加**
   - `data/drum-patterns.ts` / `data/bass-patterns.ts` / `data/backing-patterns.ts` に追加
   - パターン定義が既存の型定義に合致することを確認

3. **音声テスト**
   - `lib/audio/` の該当再生ロジックが新パターンを正しく処理することを確認
   - ブラウザで再生して音を確認

4. **UI検証**
   - パターン選択UIに新パターンが表示されることを確認
   - 選択・切り替えが正常に動作することを確認
