# 音楽用語→UIラベル対応表

本プロジェクトでは、ユーザー向けUIで音楽理論用語を直接使わず、
直感的な日本語ラベルに変換して表示する。

| 音楽理論用語 | UI表示ラベル | 補足説明（ツールチップ用） |
|-------------|-------------|-------------------------|
| トニック (Tonic) | 安定 | 曲の「家」。落ち着いた感じ |
| サブドミナント (Subdominant) | 展開 | 曲を動かす。どこかに行きたい感じ |
| ドミナント (Dominant) | 緊張 | ドキドキ感。「家」に帰りたくなる |
| ダイアトニックコード | 基本コード | このキーの標準的なコード |
| ノンダイアトニックコード | スパイスコード | ちょっと変わった響きのコード |
| セカンダリードミナント | 借用コード | 他のキーからの出張コード |
| テンションコード | おしゃれコード | 音を足してリッチにしたコード |
| 転調 (Modulation) | キーチェンジ | 曲の途中でキーが変わること |
| ドミナントモーション | 解決 | 緊張→安定の流れ |
| サブドミナントマイナー | 切ないスパイス | 一瞬マイナーを借りて切なさを出す |
| パッシングディミニッシュ | 経過コード | コードとコードの間をつなぐ |
| ペダルポイント | ベース持続 | ベース音を変えずにコードだけ動かす |
| リハーモナイゼーション | コード置き換え | 同じメロディに別のコードを当てる |
