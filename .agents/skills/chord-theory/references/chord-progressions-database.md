# コード進行データベース（度数表記）

## 定番進行

### 王道進行 (4536)
- 度数: IV → V → IIIm → VIm
- Cキー例: F → G → Em → Am
- タグ: エモい, 明るい
- 使用曲: 残酷な天使のテーゼ, 千本桜, ギルティクラウン OP

### カノン進行
- 度数: I → V → VIm → IIIm → IV → I → IV → V
- Cキー例: C → G → Am → Em → F → C → F → G
- タグ: 切ない, 明るい
- 使用曲: メルト, パッヘルベルのカノン, 卒業写真

### 小室進行
- 度数: VIm → IV → V → I
- Cキー例: Am → F → G → C
- タグ: エモい, 疾走感
- 使用曲: Get Wild, ロストワンの号哭, 残響散歌

### 丸サ進行 (Just The Two Of Us)
- 度数: IVmaj7 → III7 → VIm7 → (I7 or V/VII)
- Cキー例: Fmaj7 → E7 → Am7 → C7（or Gm7→C7）
- タグ: オシャレ, 中毒性
- 使用曲: 丸の内サディスティック, 夜に駆ける, フォニイ

### 6236進行
- 度数: VIm → IIm → IIIm → VIm
- Cキー例: Am → Dm → Em → Am
- タグ: 中毒性, 疾走感, ダーク
- 使用曲: ドーナツホール, ベノム, アスノヨゾラ哨戒班

### バラード進行 (ポップパンク進行)
- 度数: I → V → VIm → IV
- Cキー例: C → G → Am → F
- タグ: 切ない, 明るい
- 使用曲: Let It Be, ハルジオン, 前前前世

### 上昇進行

- 使用曲: 六兆年と一夜物語, 紅蓮華

### レトロ進行 (50s進行)

## 近年のヒット曲系進行

### YOASOBI風交互進行
- 度数: IV → V → IIIm → VIm → IVmaj7 → III7 → VIm7 → V
- 説明: 前半が王道進行、後半が丸サ進行に近い構造。ドラマチック
- タグ: オシャレ, エモい, 中毒性
- 使用曲: アイドル, 群青

### Vaundy風 ファンク進行
- 度数: IVmaj7 → III7 → VIm7 → IIm7 → IVmaj7 → III7 → VIm7 → V7
- タグ: オシャレ, 中毒性
- 使用曲: 怪獣の花唄, 東京フラッシュ

### ヨナ抜き風進行
- 度数: I → IIm → IV → V → I → IIm → IV → V
- 説明: ペンタトニック的な響き。和風・アジア風
- タグ: 明るい, 中毒性
- 使用曲: 千本桜(一部), 和楽器バンド系

### ダークマイナーループ
- 度数: VIm → VIm → IV → V
- Cキー例: Am → Am → F → G
- タグ: ダーク, エモい
- 使用曲: ロキ, シャルル, ヴィラン

### EDM/ボカロックドロップ

- タグ: 疾走感, 中毒性, ダーク
- 使用曲: グッバイ宣言, フォニイ, KING
